Для развертывания

в файле TaskManagementSystem\src\test\resources\application.properties нужно поменять значения
pring.datasource.url=указать url БД и схемы в ней
spring.datasource.username=пользователь к БД
spring.datasource.password=пароль пользователя к БД


в файле TaskManagementSystem\src\test\resources\application.properties нужно поменять значения
pring.datasource.url=указать url тестовой БД и схемы в ней
spring.datasource.username=пользователь к тестовой БД
spring.datasource.password=пароль пользователя к тестовой БД

У пользователя БД должны быть права на CRUD операции
Используеться Mysql
